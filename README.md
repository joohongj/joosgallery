# MEDP33100 Assignment: Online Store

Design a fictional e-commerce webpage using responsive web design principles.

An example of an online store homepage:
![shop-homepage.png](shop-homepage.png)

### **Requirements**

- You only need to create 1 homepage
- Create a responsive layout for desktop, tablet, and mobile
- Use HTML and CSS only
- Include visually appealing and user-friendly design
- Use AI-generated images if desired
- Maintain consistent typography and branding

### **Submission**

1. Fork this [repository](https://github.com/HunterCollege-MEDP33100-Spring2026/online-store.git) by clicking the "Fork" button (**Note: do this *before* you start coding!)** ![forked-repo.png](forked-repo.png)
2. Clone the forked repository through Github Desktop
3. In VS Code, add your code and push updates to your repository
3. Enable [GitHub Pages](https://www.notion.so/How-to-enable-Github-Pages-23dda96d871f80c99f2fec87680ccf07?pvs=21) (Note: your main HTML file must be named `index.html`)
4. Create a [pull request](https://www.notion.so/How-to-create-a-Pull-request-PR-2cbda96d871f81738fb6c2d48ca39c54?pvs=21) (Note: your homework is not considered submitted unless this is created!)
5. Share your page in the next class
